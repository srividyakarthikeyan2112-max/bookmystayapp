# Book My Stay - Hotel Booking Management System

A Java-based hotel booking application providing a comprehensive platform for managing hotel reservations and bookings.

## Overview

Book My Stay is a hotel booking management system designed to simplify the process of searching, booking, and managing hotel reservations. This application provides users with an intuitive interface for browsing available hotels and making reservations.

## Features

- Hotel search and browsing
- Booking management
- User-friendly interface
- Reservation tracking

## Getting Started

### Prerequisites

- Java JDK 8 or higher
- Maven or Gradle (optional, for dependency management)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/srividyakarthikeyan2112-max/bookmystayapp.git
```

2. Navigate to the project directory:
```bash
cd bookmystayapp
```

3. Compile the Java files:
```bash
javac UseCase1HotelBookingApp.java
```

### Running the Application

Execute the main class:
```bash
java UseCase1HotelBookingApp
```
